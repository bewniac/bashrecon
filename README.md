# bashrecon
Simple recon tool written in bash. 

## Installation
Developed for Ubuntu 20.04 LTS.
```
chmod +x ./install.sh && ./install.sh
```

## Docker
```
docker build -t bashrecon .
```
```
docker run -it bashrecon 
```
